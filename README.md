## Requirements

* [Node.js](https://nodejs.org/en/download/package-manager/)

## Installation

Being in the repository root directory, run:

```sh
yarn
```

## Running

Being in the repository root directory, run:

```sh
yarn start
```

## Running test (in watch mode)

Being in the repository root directory, run:

For unit-testing:

```sh
yarn test:unit
```

For e2e-testing:

```sh
yarn test:e2e
```

For unit- and e2e-testing:

```sh
yarn test
```
