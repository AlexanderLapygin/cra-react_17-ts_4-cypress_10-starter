## Requirements

* [Node.js](https://nodejs.org/en/download/package-manager/)

## Installation

Being in the repository root directory, run:

```sh
yarn
```

## Running

Being in the repository root directory, run:

```sh
yarn start
```

## Running test

Being in the repository root directory, run:

### In ui mode

```sh
yarn cypress:open
```
### In command line mode

```sh
yarn cypress:run
```
