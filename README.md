## Requirements

* [Node.js](https://nodejs.org/en/download/package-manager/)

## Installation

Being in the repository root directory, run:

```sh
yarn
```

## Running

Being in the repository root directory, run:

```sh
yarn start
```

## Running test

Being in the repository root directory, run:

### In interactive mode

```sh
yarn cypress:open
```
### Headlessly

```sh
yarn cypress:run
```
